Use widgets

(Category: Forms Modeling)

Use the set of widgets available in the Form Builder palette to define the characteristics of data fields and buttons in a form.

Associate a form field with the specific variable type it is designed to accept and handle.

Also where appropriate, define initial values (what will show up in the form when presented) and what type of variable or expression to save the field contents to when the form is submitted.

Fields defined by widgets will appear on the end user form.
