Condition

(Category: Data management)

Transitions can be configured using conditions to determine which path the flow should take.

There are several ways to define transition conditions:

  • define a transition as the default flow
  • directly enter a script into the condition field
  • use the expression editor to create a script
  • use a decision table to create a script

When a transition has been configured with a condition, the condition can be displayed as a label on the transition in the process diagram.
