Add a gate

(Category: Process Modeling)

Gateways are modeling elements that are used to control how sequence flows interact as they converge and diverge within a process.

There are 3 types of gateways available for process design in Bonita Studio: AND, XOR and Inclusive.

There are two ways to add gateways to a process design :

  • using drag and drop from the design palette
  • via the context menu
