Multi-instantiated data fields

(Category: Forms Modeling)

Multi-instantiated data fields are presented in the end user form with an “add” (and” remove”) button, allowing the end user to duplicate the same type of field multiple times, and enter data in each one.

When you configure a multi-instantiated data field (in BOS Subscription Pack), you can specify a minimum and maximum number of times a user can duplicate it. You can also multi-instantiate a group of widgets.
